# PM Mastery System - Learning Roadmap

## 2-Minute Overview
This system addresses 8 core PM skill gaps identified from real feedback, each mapped to frameworks from top product leaders. Your journey: identify your gaps, learn the framework, practice with questions, test your progress.

## Module Overview

### Module 1: Communication Clarity
Time to complete: 2-3 hours
Practice time: 5 mins/day for 2 weeks
```
01-Communication-Clarity/
├── 00-Index.md
├── 01-Answer-First.md
├── 02-Impact-Translation.md
└── 03-Self-Grading.md
```
Key Framework: Ian McAllister's "Answer First, Then Explain"
Your Gap: "Answer the exact question being asked, not adjacent ones"
Outcome: Every communication lands with clarity and impact

### Module 2: Data Storytelling
Time to complete: 3-4 hours
Practice time: 10 mins before every data review
```
02-Data-Storytelling/
├── 00-Index.md
├── 01-Data-Triangulation.md
├── 02-Baseline-First.md
└── 03-Cohort-Analysis.md
```
Key Framework: Shaun Clowes' "Data as Compass, Not GPS"
Your Gap: "No baseline metric after 6 months of working with data"
Outcome: Every data point tells a complete story with context

### Module 3: Ownership & Execution

## Recommended Learning Paths

### Path A: "I Get This Feedback Constantly" (Urgent)
Week 1: Your #1 feedback theme module
  - Read content -> Practice questions -> Daily application

Week 2: Continue + add weekly reflection test
  - Friday self-assessment -> Identify gaps -> Adjust

Week 3-4: Move to next priority theme

### Path B: "I Want Comprehensive Growth" (Thorough)
Weeks 1-2: Communication Clarity (foundation for everything)
Weeks 3-4: Data Storytelling (your data becomes weapons)
Weeks 5-6: Ownership & Execution (build PM credibility)
Weeks 7-8: Product Strategy (level up your docs)
Weeks 9-10: Leadership & Trust (make impact visible)
Weeks 11-12: Product-Led Growth (build systematic growth)

### Path C: "I Have a PRD Due" (Just-in-Time)
Today:
  - Read 04-Product-Strategy/03-PRD-Quality.md (15 mins)
  - Apply the Problem Paragraph framework
  - Use the checklist before submitting

## Success Metrics
Track your progress with these indicators:

### Leading Indicators (Weekly)
- Completed weekly reflection test
- Applied at least 1 question daily
- Received at least 1 piece of feedback

### Lagging Indicators (Monthly)
- Feedback themes are shifting (fewer repeats)
- Manager notes improvement in specific areas
- Confidence in meetings has increased
